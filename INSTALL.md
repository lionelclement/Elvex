## Requirements

To fully compile Elvex, make sure you have the following tools installed on your computer:

- g++
- flex
- bison
- libxml2-dev
- cmake

## Compilation

To compile Elvex, follow these commands:

1. Run: `cmake .`
2. Run: `make`
3. Run: `sudo make install`
